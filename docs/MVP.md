# MVP 定义

## MVP 城市选择：重庆

**选择理由**：
- 国际知名度高（火锅、山城、洪崖洞）
- 5A 景区密集（武隆、大足、巫山）
- 交通枢纽（江北国际机场、高铁网）
- 内容素材丰富（13 个景点有充足故事可写）

## MVP 核心功能（Sprint 0-4）

```
Sprint 0: 项目骨架 ✅
Sprint 1: AI 行程规划 ✅
Sprint 2: POI 数据扩展 🟡
Sprint 3: 预订与支付 ⬜
Sprint 4: AI 客服 + 部署 ✅
```

## 已上线功能清单

| 功能 | 状态 | 技术实现 |
|------|------|---------|
| 景点卡片浏览 | ✅ | Pure HTML masonry grid + Unsplash 图片 |
| 景点详情弹窗 | ✅ | Overlay panel + 图片横滑画廊 + lightbox |
| 旅行笔记 | ✅ | 12 篇，景点关联跳转 |
| AI 行程计划 | ✅ | 模板生成 + DeepSeek Function Calling |
| AI 旅行助手 | ✅ | 关键词匹配 + LLM 双模式 |
| 转人工客服 | ✅ | 表单提交 + admin 后台查看 |
| 费用预估 | ✅ | 分项报价 + 5% 服务费 |
