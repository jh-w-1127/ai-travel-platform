## 三、各景点最佳拍照机位速查

### 洪崖洞
- 📍 **千厮门大桥人行道** → 侧面全景，最经典角度
- 📍 **对岸南滨路** → 长焦拍整面悬崖灯海
- 📍 **戴家巷悬崖步道** → 纵向纵深感 + 嘉陵江入镜
- 📍 **大剧院江滩** → 低角度 + 水面倒影（退潮时）
- ⏰ 亮灯时间：夏季 20:00-23:00 / 冬季 18:30-23:00

### 大足石刻
- 📍 **宝顶山千手观音** → 正面仰拍
- 📍 **圆觉洞** → 自然光透入时（上午10点最佳）
- 📍 **吹笛女** → 侧面拍表情细节

### 武隆天生三桥
- 📍 **天龙桥正下方** → 广角仰拍
- 📍 **天福官驿门口** → 电影《满城尽带黄金甲》同款
- 📍 **青龙桥** → 6月/12月正午"光柱"时刻

### 白帝城·瞿塘峡
- 📍 **白帝城观景台** → 夔门正面 + 10元人民币同款
- 📍 **三峡之巅** → 高空俯拍夔门

### 李子坝
- 📍 **李子坝观景平台** → 正下方仰拍穿楼
- 📍 **牛角沱方向** → 侧面拍列车"钻"进楼体

## 四、图片风格建议（对应文档中的设计规范）

| 用途 | 风格 | 来源建议 |

| 首页 Hero | 高动态范围夜景，1920×1080+ | Unsplash "Chongqing night skyline" |
